# Roadmap

The v1 release is a research-only, fixture-first foundation.

- Add opt-in live filings for HKEXnews, EDINET/TDnet, SSE/SZSE, and European
  issuer IR feeds with jurisdiction-specific parsers.
- Add a PostgreSQL-backed analysis archive and signed, immutable report links.
- Add licensed delayed/real-time market-data adapters where redistribution
  terms permit it.
- Expand accounting reconciliation, ADR ratio handling, corporate actions, and
  trading calendars.
- Add user-defined assumption workbooks and richer valuation sensitivity grids.
- Add accessible PNG export and optional server-rendered report PDFs.

No brokerage connection or automated trade execution is planned for the core
project.
